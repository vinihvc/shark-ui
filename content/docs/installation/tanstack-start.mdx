---

### Create project

To create a new TanStack Start project, you can use the following command:

```bash
npm create @tanstack/start@latest --tailwind --add-ons shadcn
```

### Add Components

You can now start adding components to your project.

```bash
pnpm dlx shadcn@latest add https://shark.vini.one/r/button.json
```

The command above will add the `Button` component to your project. You can then import it like this:

```tsx title="app/routes/index.tsx" lineNumbers
import { Button } from "@/components/ui/button" 

function App() {
  return ( 
    <div> 
      <Button>Click me</Button> 
    </div> 
  )
} 
```
