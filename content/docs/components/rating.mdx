## API Reference

### Rating

Root element of the rating. Renders a `div` and forwards props to Ark UI's RatingGroup.Root.

| Prop | Type | Default |
|------|------|---------|
| `allowHalf` | `boolean` | `false` |
| `count` | `number` | `5` |
| `defaultValue` | `number` | `-` |
| `value` | `number` | `-` |
| `onValueChange` | `(details: ValueChangeDetails) => void` | `-` |
| `readOnly` | `boolean` | `-` |
| `disabled` | `boolean` | `-` |
| `className` | `string` | `-` |

---

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/rating-group#api-reference).
