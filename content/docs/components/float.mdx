## Usage

Float requires a parent with `position: relative`. Place `<Float />` as a sibling of the anchored content.

```tsx
import { Float } from "@/components/ui/float";

<div className="relative">
  <Float placement="top-end">
    {/** floated content **/}
  </Float>
</div>
```
