## Usage

```tsx
import { 
  Field, 
  FieldLabel, 
  FieldInput, 
  FieldHelper, 
  FieldError 
} from "@/components/ui/field";
import { Input } from "@/components/ui/input";
```

```tsx showLineNumbers
<Field>
  <FieldLabel>Email</FieldLabel>
  <Input type="email" placeholder="Your best email address" />
  <FieldHelper>Enter your email address</FieldHelper>
  <FieldError>Email is required</FieldError>
</Field>
```

## Validation and Errors

- Add `invalid` to Field to switch the entire block into an error state.
- Add `invalid` on the input itself to switch the input into an error state.

```tsx
<Field invalid>
  <FieldLabel>Email</FieldLabel>
  <Input type="email" />
  <FieldError>Enter a valid email address.</FieldError>
</Field>
```

## Accessibility

- `<FieldSet />` and `<FieldLegend />` keep related controls grouped for keyboard and assistive tech users.
- `<Field />` outputs `role="group"` so nested controls inherit labeling from `<FieldLabel />` and `<FieldLegend />` when combined.
- Apply `<FieldSeparator />` sparingly to ensure screen readers encounter clear section boundaries.

## API Reference

### Field

Root component. Flex container for label, control, and helper/error text. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` |
| `reverse` | `boolean` | `false` |
| `invalid` | `boolean` | `-` |
| `disabled` | `boolean` | `-` |
| `required` | `boolean` | `-` |
| `readOnly` | `boolean` | `-` |
| `asChild` | `boolean` | `-` |
| `className` | `string` | `-` |

### FieldSet

Groups related form controls. Renders a `fieldset`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### FieldLegend

Legend for a FieldSet. Renders a `legend`.

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `"legend" \| "label"` | `"legend"` |
| `className` | `string` | `-` |

### FieldGroup

Container for a group of fields. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### FieldContent

Wrapper for label + description/error. Use with horizontal orientation. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### FieldLabel

Label for the field. Associates with the control via `htmlFor` / `id`. Renders a `label`.

| Prop | Type | Default |
| --- | --- | --- |
| `asChild` | `boolean` | `-` |
| `className` | `string` | `-` |

### FieldLabelRequired

Required indicator (e.g. asterisk). Use inside FieldLabel. Renders a `span`.

| Prop | Type | Default |
| --- | --- | --- |
| `children` | `ReactNode` | `"*"` |
| `className` | `string` | `-` |

### FieldTitle

Title wrapper for field content. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### FieldDescription

Description or helper text for the field. Muted foreground. Renders a `p`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### FieldSeparator

Horizontal separator between field groups. Optional center text. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `children` | `ReactNode` | `-` |
| `className` | `string` | `-` |

### FieldHelper

Helper text (alias for description). Renders a `p`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### FieldError

Error message for invalid state. Destructive foreground. Renders a `p`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

---

For a complete list of props, see the [Ark UI Field](https://ark-ui.com/docs/components/field#api-reference) and [Fieldset](https://ark-ui.com/docs/components/fieldset#api-reference) documentation.
