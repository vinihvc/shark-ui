pnpm add @ark-ui/react tailwind-variants

## Usage

```tsx lineNumbers
import { 
  ButtonGroup,
  ButtonGroupText
  ButtonGroupSeparator,
} from "@/components/ui/button-group";
```

```tsx lineNumbers
<ButtonGroup> 
  <Button />
  <ButtonGroupSeparator />
  <Button />
</ButtonGroup>
```

## Accessibility

Use `aria-label` attribute to provide a label for the button group.


## Orientation

You can change the orientation of the button group using the `orientation` prop.

Available orientations are: `horizontal`, `vertical`.

<ComponentVariants variants={['horizontal', 'vertical']} defaultValue="horizontal">
  <TabsContent value="horizontal">
    <ButtonGroup>
      <Button clickEffect={false} variant="outline" size="icon-md">
        <SkipBack />
      </Button>

      <Button clickEffect={false} variant="outline" size="icon-md">
        <Play />
      </Button>

      <Button clickEffect={false} variant="outline" size="icon-md">
        <SkipForward />
      </Button>
    </ButtonGroup>
  </TabsContent>

  <TabsContent value="vertical">
    <ButtonGroup orientation="vertical">
      <Button clickEffect={false} variant="outline" size="icon-md">
        <Plus />
      </Button>

      <Button clickEffect={false} variant="outline" size="icon-md">
        <Minus />
      </Button>
    </ButtonGroup>
  </TabsContent>
</ComponentVariants>

## Separator

You can add a separator between the buttons using the `ButtonGroupSeparator`.

<ComponentPreview code={`<ButtonGroup>
    <Button variant="secondary" size="icon-md">
      <Plus />
    </Button>
    
    <ButtonGroupSeparator />
    
    <Button variant="secondary" size="icon-md">
      <Minus />
    </Button>
</ButtonGroup>
`}>
  <ButtonGroup>
    <Button clickEffect={false} variant="secondary" size="icon-md">
      <Minus />
    </Button>
    
    <ButtonGroupSeparator />
    
    <Button clickEffect={false} variant="secondary" size="icon-md">
      <Plus />
    </Button>
  </ButtonGroup>
</ComponentPreview>


## Text

You can add text between the buttons using the `ButtonGroupText` component.

<ComponentPreview code={`<ButtonGroup>

    </Button>

`}>

    <ButtonGroupText>Create user</ButtonGroupText>
    <Button clickEffect={false} variant="secondary" size="icon-md">
      <Plus />
    </Button>
  </ButtonGroup>
</ComponentPreview>
