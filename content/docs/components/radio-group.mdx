## Usage

```tsx lineNumbers
import {
  RadioGroup,
  RadioGroupItem,
} from "@/components/ui/radio-group";
```

```tsx lineNumbers
<RadioGroup>
  <RadioGroupItem value="1">Option 1</RadioGroupItem>
  <RadioGroupItem value="2">Option 2</RadioGroupItem>
  <RadioGroupItem value="3">Option 3</RadioGroupItem>
</RadioGroup>
```

## Invalid State

Use the `invalid` prop on `<Field />` to indicate an invalid state.

<ComponentPreview componentName="radio-group" fileName="example-invalid" />

You can pass `invalid` to the `<Field />` to style the label and description or to individual `<RadioGroupItem />` for item-level validation.

Use `<RadioGroup />` with `<FieldSet />` and `<FieldLegend />` to group options with a legend.

<ComponentPreview componentName="radio-group" fileName="example-radio-group" />

Use `defaultValue` to set the default selected value.

```tsx lineNumbers

  <RadioGroupItem value="default" />
  <RadioGroupItem value="comfortable" />
  <RadioGroupItem value="compact" />
</RadioGroup>
```

## API Reference

### RadioGroup

Root element of the radio group. Renders a `div`

| Prop | Type | Default |
|------|------|---------|
| `value` | `string` | `-` |
| `defaultValue` | `string` | `-` |
| `disabled` | `boolean` | `false` |
| `name` | `string` | `-` |
| `orientation` | `'horizontal' \| 'vertical'` | `-` |
| `onValueChange` | `(details: ValueChangeDetails) => void` | `-` |
| `className` | `string` | `-` |

### RadioGroupItem

A single radio item. Renders a `label`

| Prop | Type | Default |
|------|------|---------|
| `value` | `string` | `-` |
| `disabled` | `boolean` | `false` |
| `invalid` | `boolean` | `false` |
| `className` | `string` | `-` |
