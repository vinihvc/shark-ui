## Usage

```tsx
import {
  RadioGroup,
  RadioGroupItem,
} from "@/components/ui/radio-group";
```

```tsx showLineNumbers
<RadioGroup>
  <RadioGroupItem value="item-1">{/* label */}</RadioGroupItem>
  <RadioGroupItem value="item-2">{/* label */}</RadioGroupItem>
</RadioGroup>
```

## API Reference

### RadioGroup

Root element of the radio group. Renders a `div`

| Prop | Type | Default |
|------|------|---------|
| `value` | `string` | `-` |
| `defaultValue` | `string` | `-` |
| `disabled` | `boolean` | `false` |
| `name` | `string` | `-` |
| `onValueChange` | `(details: ValueChangeDetails) => void` | `-` |
| `className` | `string` | `-` |

### RadioGroupItem

A single radio item. Renders a `label`

| Prop | Type | Default |
|------|------|---------|
| `value` | `string` | `-` |
| `disabled` | `boolean` | `false` |
| `invalid` | `boolean` | `false` |
| `className` | `string` | `-` |

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/radio-group#api-reference).
