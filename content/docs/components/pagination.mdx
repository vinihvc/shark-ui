## Usage

```tsx
import {
  Pagination,
  PaginationItems,
  PaginationNext,
  PaginationPrevious,
} from "@/components/ui/pagination";
```

```tsx showLineNumbers
<Pagination>
  <PaginationPrevious />
  <PaginationItems />
  <PaginationNext />
</Pagination>
```

## API Reference

### Pagination

| Prop | Type | Default |
| --- | --- | --- |
| `count` | `number` | - |
| `page` | `number` | - |
| `defaultPage` | `number` | `1` |
| `pageSize` | `number` | - |
| `defaultPageSize` | `number` | `10` |
| `siblingCount` | `number` | `1` |
| `onPageChange` | `(details: PageChangeDetails) => void` | - |
| `onPageSizeChange` | `(details: PageSizeChangeDetails) => void` | - |

---

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/pagination#api-reference).
