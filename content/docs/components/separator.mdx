## API Reference

### Separator

Horizontal or vertical divider between content.

| Prop | Type | Default | 
| --- | --- | --- | 
| `orientation` | `"horizontal"`, `"vertical"` | `"horizontal"` |
| `className` | `string` | `-` |

For complete list of props, see [Ark UI documentation](https://ark-ui.com/docs/components/separator#api-reference).
