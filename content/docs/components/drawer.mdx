## Usage

Wrap your app (or layout) with `DrawerProvider` to enable the indent effect—when a drawer opens, the page content scales down and gets rounded corners.

```tsx
import { DrawerProvider } from "@/components/ui/drawer";

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        <DrawerProvider>{children}</DrawerProvider>
      </body>
    </html>
  );
}
```

```tsx
import {
  Drawer,
  DrawerTrigger,
  DrawerContent,
  DrawerHeader,
  DrawerTitle,
  DrawerDescription,
  DrawerBody,
  DrawerFooter,
  DrawerClose,
} from "@/components/ui/drawer";
```

```tsx showLineNumbers
<Drawer swipeDirection="down">
  <DrawerTrigger asChild>
    <Button>Open</Button>
  </DrawerTrigger>
  <DrawerContent>
    <DrawerHeader title="Title" description="Description" />
    <DrawerBody>
      {/** your content here */}
    </DrawerBody>
    <DrawerFooter>
      <DrawerClose asChild>
        <Button>Close</Button>
      </DrawerClose>
    </DrawerFooter>
  </DrawerContent>
</Drawer>
```

## API Reference

### DrawerProvider

Provider that enables the indent effect. Wrap your app or layout with it. When a drawer opens, the page content inside scales down and gets rounded corners. Renders `Drawer.Stack` with `IndentBackground` and `Indent` parts.

| Prop | Type | Default |
|------|------|---------|
| `className` | `string` | `-` |

### Drawer

Root element of the drawer. Renders a drawer context. Uses Ark UI Drawer (zag-js) under the hood.

| Prop | Type | Default |
|------|------|---------|
| `open` | `boolean` | `-` |
| `defaultOpen` | `boolean` | `false` |
| `onOpenChange` | `({ open }: OpenChangeDetails) => void` | `-` |
| `swipeDirection` | `"up" \| "down" \| "left" \| "right"` | `"down"` |
| `snapPoints` | `(number \| string)[]` | `[1]` |
| `defaultSnapPoint` | `number \| string \| null` | `1` |
| `snapToSequentialPoints` | `boolean` | `false` |
| `closeThreshold` | `number` | `0.25` |
| `swipeVelocityThreshold` | `number` | `700` |
| `preventDragOnScroll` | `boolean` | `true` |
| `modal` | `boolean` | `true` |
| `closeOnEscape` | `boolean` | `true` |
| `closeOnInteractOutside` | `boolean` | `true` |
| `className` | `string` | `-` |

### DrawerTrigger

Button that opens the drawer. Renders a `button`.

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | `-` |

### DrawerContent

Container for the drawer content. Renders a `div` with swipe support. Direction is inherited from `swipeDirection` on the root.

| Prop | Type | Default |
|------|------|---------|
| `variant` | `"default" \| "inset"` | `"default"` |
| `className` | `string` | `-` |

### DrawerHeader

Header container for the drawer. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `title` | `string` | `-` |
| `description` | `string` | `-` |
| `className` | `string` | `-` |

### DrawerBody

Scrollable content container. Renders a `div` wrapped in a `ScrollArea`.

| Prop | Type | Default |
|------|------|---------|
| `scrollFade` | `boolean` | `false` |
| `className` | `string` | `-` |

### DrawerFooter

Footer container for the drawer. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `className` | `string` | `-` |

### DrawerTitle

Title element for the drawer. Renders an `h2`.

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | `-` |

### DrawerDescription

Description element for the drawer. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | `-` |

### DrawerClose

Button that closes the drawer. Renders a `button`.

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | `-` |

### DrawerBackdrop

Backdrop overlay for the drawer. Renders a `div`. Used internally by `DrawerContent`.

| Prop | Type | Default |
|------|------|---------|
| `className` | `string` | `-` |

### DrawerPositioner

Positioning wrapper for the drawer content. Renders a `div`. Used internally by `DrawerContent`.

| Prop | Type | Default |
|------|------|---------|
| `variant` | `"default" \| "inset"` | `"default"` |
| `className` | `string` | `-` |
