## API Reference

### Input

Text input field. Renders an `input`. Accepts all native input attributes.

| Prop | Type | Default |
| --- | --- | --- |
| `size` | `sm` \| `md` \| `lg` | `md` |
| `type` | `string` | `text` |
| `className` | `string` | `-` |

---

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/input#api-reference).
