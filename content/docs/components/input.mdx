## Examples

### Field

Use `Field`, `FieldLabel` and `FieldDescription` to create a input with a label and a description.

<ComponentPreview componentName="input" fileName="example-field" />

### Field Group

Use `FieldGroup` to show multiple `Field` blocks and to build forms.

<ComponentPreview componentName="input" fileName="example-field-group" />

### Disabled

Use the `disabled` prop to disable the input. To style the disabled state, add the `disabled` attribute to the `<Field />` component.

<ComponentPreview componentName="input" fileName="example-disabled" />

### Invalid

Use the `aria-invalid` prop to mark the input as invalid. Pass the `invalid` prop to `<Field />` to disable the label and `<Input />`, or to `<Input />` alone.

<ComponentPreview componentName="input" fileName="example-invalid" />

### File

Use the `type="file"` prop to create a file input.

The <Link href="/docs/components/file-upload">`<FileUpload />`</Link> component handles file uploads.

<ComponentPreview componentName="input" fileName="example-file" />

### Inline

Use the `orientation="horizontal"` prop to create a inline input.

<ComponentPreview componentName="input" fileName="example-inline" />

### Grid

<ComponentPreview componentName="input" fileName="example-grid" />

### Required

Use the `required` prop to mark the input as required. Pass `required` to `<Field />` to make the `<Input />` element required.

 Use `<FieldLabelRequired />` to add a required indicator to the label.

<ComponentPreview componentName="input" fileName="example-required" />
