## Usage

```tsx

  Announcement,
  AnnouncementTag,
  AnnouncementTitle,
} from "@/components/ui/announcement";
```

```tsx showLineNumbers

  <AnnouncementTitle>New feature added</AnnouncementTitle>
</Announcement>
```

## API Reference

### Announcement

Root element of the announcement. Renders a `div`

| Prop | Type | Default |
| --- | --- | --- |
| `role` | `status`, `alert`, or other ARIA live region role | `status` |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### AnnouncementTag

Optional label for the announcement. Renders a `span`. Uses Badge variants. Icons in the tag inherit the variant color.

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `default`, `secondary`, `outline`, `success`, `info`, `warning`, `destructive` | `default` |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### AnnouncementTitle

Main title content of the announcement. Renders a `span`. Icons in the title use `text-muted-foreground`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |
