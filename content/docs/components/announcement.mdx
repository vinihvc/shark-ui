## API Reference

### Announcement

Root wrapper for announcement banners or notices.

| Prop | Type | Default |
| --- | --- | --- |
| `role` | `status`, `alert`| `status` |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### AnnouncementTitle

Main title or message of the announcement.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |
