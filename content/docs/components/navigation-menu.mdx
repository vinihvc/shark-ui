## Usage

```tsx
import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuItem,
  NavigationMenuLink,
  NavigationMenuList,
  NavigationMenuTrigger,
} from "@/components/ui/navigation-menu";
```

```tsx showLineNumbers
<NavigationMenu>
  <NavigationMenuList>
    <NavigationMenuItem value="products">
      <NavigationMenuTrigger>Products</NavigationMenuTrigger>
      <NavigationMenuContent>
        <NavigationMenuLink value="products" href="/analytics">
          Analytics
        </NavigationMenuLink>
      </NavigationMenuContent>
    </NavigationMenuItem>
    <NavigationMenuItem value="pricing">
      <NavigationMenuLink value="pricing" href="/pricing">
        Pricing
      </NavigationMenuLink>
    </NavigationMenuItem>
  </NavigationMenuList>
</NavigationMenu>
```

## Patterns

### Basic pattern

The basic pattern places content directly within each item. Each dropdown appears below its trigger. Use this for simple navigation menus.

<ComponentPreview componentName="navigation-menu" fileName="example-default" />

### Viewport pattern

The viewport pattern uses a shared viewport container for all content. This enables smooth transitions between dropdowns and is ideal for header navigation bars. Include `NavigationMenuViewportPositioner`, `NavigationMenuViewport`, and `NavigationMenuIndicator` when using this pattern.

<ComponentPreview componentName="navigation-menu" fileName="example-viewport" />

## Controlled

Use the `value` and `onValueChange` props to control which item is open programmatically.

## API Reference

### NavigationMenu

Root element. Renders a `nav`

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | — |
| `defaultValue` | `string` | — |
| `onValueChange` | `(details: { value: string }) => void` | — |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |
| `openDelay` | `number` | `200` |
| `closeDelay` | `number` | `300` |

### NavigationMenuList

List container for menu items. Renders a `div`

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | — |

### NavigationMenuItem

Container for a single item. Renders a `div`. Requires `value`

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | **required** |
| `disabled` | `boolean` | `false` |

### NavigationMenuTrigger

Button that opens dropdown content. Renders a `button`

| Prop | Type | Default |
| --- | --- | --- |
| `disabled` | `boolean` | — |

### NavigationMenuContent

Dropdown content. Renders a `div`

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | — |

### NavigationMenuLink

Link for navigation. Renders an `a`. Use for direct links or links inside content.

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | — |
| `href` | `string` | — |
| `current` | `boolean` | `false` |
| `closeOnClick` | `boolean` | `true` |

### NavigationMenuViewportPositioner / NavigationMenuViewport

Used with the viewport pattern. Positioner wraps the viewport; content is portaled into the viewport.

### NavigationMenuIndicator

Animated indicator that follows the active trigger. Use inside `NavigationMenuList` when using the viewport pattern.

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `"default" \| "underline"` | `"default"` |
