## Usage

The frame is a compound component for grouping related information with a header, content panels, and optional footer.

```tsx
import {
  Frame,
  FrameHeader,
  FrameTitle,
  FrameDescription,
  FramePanel,
  FrameFooter,
} from "@/components/ui/frame";
```

```tsx showLineNumbers
<Frame>
  <FrameHeader>
    <FrameTitle>Title</FrameTitle>
    <FrameDescription>Description</FrameDescription>
  </FrameHeader>
  <FramePanel>Content</FramePanel>
  <FrameFooter>Footer</FrameFooter>
</Frame>
```

## Components

| Component | Description |
| --- | --- |
| `Frame` | Root container. Accepts `className` and standard div props. |
| `FrameHeader` | Header section containing title and description. |
| `FrameTitle` | Title text for the frame header. |
| `FrameDescription` | Description text for the frame header. |
| `FramePanel` | Panel container for frame content. Use multiple for separated sections. |
| `FrameFooter` | Footer section for the frame. |
