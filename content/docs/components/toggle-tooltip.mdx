```tsx showLineNumbers
  </ToggleTooltipTrigger>
  <ToggleTooltipContent>
    Hover or click to see this content.
  </ToggleTooltipContent>

```

## Accessibility

- **Icon-only triggers** — When the trigger is an icon without visible text, add an `aria-label` so screen readers can announce its purpose.
- **Touch and keyboard** — Unlike tooltips, Toggle Tooltip opens on click as well as hover, so touch users and keyboard users can access the content.
- **Screen reader exposure** — Content is exposed to assistive technologies. Use Toggle Tooltip when the information is essential, not decorative.
- **Escape to close** — Press Escape to dismiss the content when it is open.

## When to use

### Use Tooltip when

- The trigger has its own purpose and the popup is **supplementary**.
- The content is **non-essential**—nice-to-have clarity for sighted mouse/keyboard users.
- You're okay with **touch users not seeing it**—tooltips only open on hover/focus, not on tap.
- The content is **short and non-interactive** (no links or buttons inside).

### Use Toggle Tooltip when

- The trigger's **primary purpose** is to reveal the content.
- The content is **essential** and should be accessible to touch users and screen readers.
- You need **click-to-open** so everyone can access it—no hover required.
- The content may be **interactive** (links, buttons) since Toggle Tooltip is a popover under the hood.

### Quick reference

| Component | Trigger purpose | Touch access | Content type |
| --- | --- | --- | --- |
| **Tooltip** | Unrelated (trigger does something else) | No | Short, non-interactive |
| **Toggle Tooltip** | To open the popup | Yes | Supplementary, can be interactive |
| **Popover** | To open the popup | Yes | Rich content, headers, footers |

## API Reference

### ToggleTooltip

Root component. Manages open state, positioning, and hover behavior.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `positioning` | `PositioningOptions` | `{ placement: "top" }` | Options used to position the content |
| `openOnHover` | `boolean` | `true` | Open on hover (desktop convenience) |
| `openDelay` | `number` | `0` | Delay before opening on hover (ms) |
| `closeDelay` | `number` | `100` | Delay before closing on hover leave (ms) |
| `modal` | `boolean` | `false` | Whether to trap focus when open |
| `lazyMount` | `boolean` | `true` | Whether to enable lazy mounting |
| `unmountOnExit` | `boolean` | `true` | Whether to unmount on exit |
| `open` | `boolean` | — | The controlled open state |
| `onOpenChange` | `(details: OpenChangeDetails) => void` | — | Called when the open state changes |

### ToggleTooltipTrigger

The element that triggers the popover on click or hover. Renders a `button` by default.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `asChild` | `boolean` | — | Use the provided child element as the default rendered element |

### ToggleTooltipContent

The content displayed. Renders a `div` element with tooltip-like styling.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | — | The class name for the content |
| `asChild` | `boolean` | — | Use the provided child element as the default rendered element |

### ToggleTooltipArrow

Optional arrow pointing to the trigger. Place inside `ToggleTooltipContent`.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `asChild` | `boolean` | — | Use the provided child element as the default rendered element |
