## Usage

```tsx
import {
  ActionBar,
  ActionBarBody,
  ActionBarClose,
  ActionBarContent,
  ActionBarSelectionTrigger,
  ActionBarSeparator,
  ActionBarTrigger,
} from "@/components/ui/action-bar";
```

```tsx
<ActionBar>
  <ActionBarTrigger />
  <ActionBarContent>
    <ActionBarClose />
    <ActionBarSelectionTrigger count={3} />
    <ActionBarBody>
      {/* Your action buttons */}
    </ActionBarBody>
  </ActionBarContent>
</ActionBar>
```

## API Reference

### ActionBar

Root context provider for the action bar. Renders a `React.Fragment`.

| Prop | Type | Default |
| --- | --- | --- |
| `open` | `boolean` | - |
| `defaultOpen` | `boolean` | `false` |
| `positioning` | `{ placement?: "bottom" \| "bottom-start" \| "bottom-end", offset?: string }` | `{ placement: "bottom", offset: "16px" }` |
| `lazyMount` | `boolean` | `true` |
| `unmountOnExit` | `boolean` | `true` |
| `onOpenChange` | `(open: boolean) => void` | - |

### ActionBarContent

Floating toolbar. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `aria-label` | `string` | `"Selection actions"` |
| `aria-labelledby` | `string` | - |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |

| Attributes | Description |
| --- | --- |
| `data-slot` | `"action-bar-content"` |
| `data-state` | `"open" \| "closed"` |
| `--offset` | `"16px"` |

### ActionBarTrigger

Button that opens the action bar. Renders a `button`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |

### ActionBarClose

Button that closes the action bar. Renders a `button`.

| Prop | Type | Default |
| --- | --- | --- |
| `aria-label` | `string` | `"Close"` |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |

### ActionBarSelectionTrigger

Displays selection count or custom label. Renders a `span`.

| Prop | Type | Default |
| --- | --- | --- |
| `count` | `number` | `0` |
| `label` | `string` | - |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |

### ActionBarBody

Container for action buttons. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |
