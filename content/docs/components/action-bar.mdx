## Usage

```tsx
import {
  ActionBar,
  ActionBarClose,
  ActionBarContent,
  ActionBarSelectionTrigger,
  ActionBarSeparator,
  ActionBarTrigger,
} from "@/components/ui/action-bar";
```

```tsx
<ActionBar>
  <ActionBarTrigger />
  <ActionBarContent>
    <ActionBarClose />
    <ActionBarSelectionTrigger count={3} />
    {/* Your actions */}
  </ActionBarContent>
</ActionBar>
```

## API Reference

### ActionBar

| Prop | Type | Default |
|------|------|---------|
| `open` | `boolean` | - |
| `defaultOpen` | `boolean` | `false` |
| `lazyMount` | `boolean` | `true` |
| `unmountOnExit` | `boolean` | `true` |
| `onOpenChange` | `(open: boolean) => void` | - |

### ActionBarPositioning

Object to configure action bar placement and offset.

| Property | Type | Default |
|----------|------|---------|
| `placement` | `"bottom" \| "bottom-start" \| "bottom-end"` | `"bottom"` |
| `offset` | `number` | `16` |

### ActionBarContent

| Prop | Type | Default |
|------|------|---------|
| `positioning` | `ActionBarPositioning` | `{ placement: "bottom", offset: 16 }` |
| `aria-label` | `string` | `"Selection actions"` |
| `aria-labelledby` | `string` | - |
| `className` | `string` | - |

### ActionBarPositioner

| Prop | Type | Default |
|------|------|---------|
| `positioning` | `ActionBarPositioning` | `{ placement: "bottom", offset: 16 }` |
| `className` | `string` | - |

### ActionBarTrigger

Opens the action bar when clicked. Use `asChild` to merge props with a child element (e.g. `Button`).

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | - |

### ActionBarClose

Closes the action bar when clicked. Use `asChild` to merge close behavior with a custom element.

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | - |

### ActionBarSelectionTrigger

Displays "X items selected" when `count` is provided, or custom `label`/`children`.

| Prop | Type | Default |
|------|------|---------|
| `count` | `number` | `0` |
| `label` | `string` | - |
| `className` | `string` | - |

### ActionBarSeparator

| Prop | Type | Default |
|------|------|---------|
| `className` | `string` | - |
