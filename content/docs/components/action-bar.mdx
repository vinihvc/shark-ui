## Usage

The action bar is designed to be controlled by table or checkbox selections. Use `open` and `onOpenChange` to control visibility.

```tsx
import {
  ActionBarRoot,
  ActionBarPositioner,
  ActionBarContent,
  ActionBarCloseTrigger,
  ActionBarSelectionTrigger,
  ActionBarSeparator,
} from "@/components/ui/action-bar";

const [open, setOpen] = useState(false);

<ActionBarRoot open={open} onOpenChange={setOpen}>
  <ActionBarPositioner placement="bottom">
    <ActionBarContent>
      <ActionBarCloseTrigger />
      <ActionBarSeparator />
      <ActionBarSelectionTrigger count={3} />
      {/* Your actions */}
    </ActionBarContent>
  </ActionBarPositioner>
</ActionBarRoot>
```

## API Reference

### ActionBarRoot

| Prop | Type | Default |
|------|------|---------|
| `open` | `boolean` | - |
| `defaultOpen` | `boolean` | `false` |
| `onOpenChange` | `(open: boolean) => void` | - |

### ActionBarPositioner

| Prop | Type | Default |
|------|------|---------|
| `placement` | `"bottom" \| "bottom-start" \| "bottom-end"` | `"bottom"` |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` |

### ActionBarContent

| Prop | Type | Default |
|------|------|---------|
| `size` | `"sm" \| "md" \| "lg"` | `"md"` |

### ActionBarCloseTrigger

Closes the action bar when clicked.

### ActionBarSelectionTrigger

| Prop | Type | Default |
|------|------|---------|
| `count` | `number` | `0` |
| `label` | `string` | - |

Displays "X items selected" when `count` is provided, or custom `label`/`children`.
