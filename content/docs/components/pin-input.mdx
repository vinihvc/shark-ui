## Usage

```tsx lineNumbers
import { 
  PinInput, 
  PinInputSeparator, 
  PinInputSlot, 
} from "@/components/ui/pin-input";
```

```tsx lineNumbers
<PinInput>
  <PinInputSlot index={0} />
  <PinInputSlot index={1} />
  <PinInputSlot index={2} />
  <PinInputSeparator />
  <PinInputSlot index={3} />
  <PinInputSlot index={4} />
  <PinInputSlot index={5} />
</PinInput>
```

## API Reference

### PinInput (Root)

Extends [Ark UI PinInput.Root](https://ark-ui.com/docs/components/pin-input#api-reference) props.

| Prop | Type | Default |
| --- | --- | --- |
| `withFakeCaret` | `boolean` | `false` |
| `value` | `string` | — |
| `defaultValue` | `string` | — |
| `onValueChange` | `(details: ValueChangeDetails) => void` | — |
| `disabled` | `boolean` | — |
| `invalid` | `boolean` | — |
| `otp` | `boolean` | `true` |
| `blurOnComplete` | `boolean` | — |
| `mask` | `boolean` | — |

### PinInputSlot

Renders a single pin input field. Use `index` to specify the slot position (0-based).

| Prop | Type | Default |
| --- | --- | --- |
| `index` | `number` | **required** |

### PinInputSeparator

Renders a visual separator between slots (e.g. dash). Extends native `<hr>` props.

---

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/pin-input#api-reference).
