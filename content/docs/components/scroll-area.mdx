## API Reference

### ScrollArea

Root component. Styled wrapper for `ScrollArea.Root` from Ark UI with custom scrollbar styling.

| Prop | Type | Default | 
| --- | --- | --- | 
| direction | `vertical`, `horizontal`, `both` | `vertical` | 
| scrollFade | `boolean` | `false` | 

---

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/scroll-area#api-reference).
