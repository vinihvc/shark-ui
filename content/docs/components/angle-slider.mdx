## API Reference

### AngleSlider

Root element of the angle slider. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `value` | `number` | - |
| `defaultValue` | `number` | `0` |
| `onValueChange` | `(details: ValueChangeDetails) => void` | - |
| `step` | `number` | `1` |
| `disabled` | `boolean` | - |
| `size` | `number` | `120` |
| `thickness` | `number` | `6` |
| `markers` | `boolean \| number[]` | - |
| `markersAtSteps` | `boolean` | `false` |
| `className` | `string` | - |

### AngleSliderValue

Displays the current value. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `prefix` | `React.ReactNode \| string` | - |
| `suffix` | `React.ReactNode \| string` | - |
| `className` | `string` | - |
