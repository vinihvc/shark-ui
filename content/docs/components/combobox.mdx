## Usage

Pass an `items` array and use `ComboboxList` with a render prop. Filtering is handled internally. No type annotations needed for the common `{ label, value }` shape. For custom item shapes, type the callback parameter: `{(item: MyItemType) => ...}`.

```tsx
import {
  Combobox,
  ComboboxControl,
  ComboboxInput,
  ComboboxTrigger,
  ComboboxClearTrigger,
  ComboboxContent,
  ComboboxItem,
  ComboboxList,
} from "@/components/ui/combobox";

const items = [
  { label: "Apple", value: "apple" },
  { label: "Banana", value: "banana" },
  { label: "Cherry", value: "cherry" },
];

<Combobox items={items}>
  <ComboboxControl>
    <ComboboxInput />
    <ComboboxClearTrigger />
    <ComboboxTrigger />
  </ComboboxControl>
  <ComboboxContent>
    <ComboboxList>
      {(item) => (
        <ComboboxItem item={item} key={item.value}>
          {item.label}
        </ComboboxItem>
      )}
    </ComboboxList>
  </ComboboxContent>
</Combobox>
```

## API Reference

### Combobox

Root component. Use with `useListCollection` and `useFilter` for filtering.

| Prop | Type | Default |
| --- | --- | --- |
| `collection` | `ListCollection<T>` | **required** |
| `value` | `string \| string[]` | - |
| `defaultValue` | `string \| string[]` | - |
| `onValueChange` | `(details: ValueChangeDetails<T>) => void` | - |
| `onInputValueChange` | `(details: InputValueChangeDetails) => void` | - |
| `multiple` | `boolean` | `false` |
| `disabled` | `boolean` | - |
| `openOnClick` | `boolean` | `true` |
| `lazyMount` | `boolean` | `true` |
| `unmountOnExit` | `boolean` | `true` |
| `className` | `string` | - |

### ComboboxControl

Optional wrapper for custom layouts. Use `ComboboxInput` for the standard input-with-trigger layout.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `asChild` | `boolean` | - |

### ComboboxInput

Input with dropdown trigger and optional clear button. Renders a `<input>` and a `<button>`.

| Prop | Type | Default |
| --- | --- | --- |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` |
| `showTrigger` | `boolean` | `true` |
| `showClear` | `boolean` | `false` |
| `disabled` | `boolean` | - |
| `className` | `string` | - |

### ComboboxTrigger

Button that opens the dropdown. Renders a `button`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `asChild` | `boolean` | - |

### ComboboxClear

Button to clear the input value. 

| Prop | Type | Default |
| --- | --- | --- |
| `asChild` | `boolean` | - |

### ComboboxContent

Popup container for options. Renders a `div`

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `asChild` | `boolean` | - |

### ComboboxList

List container for combobox items. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `asChild` | `boolean` | - |

### ComboboxItem

Individual option. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `item` | `T` | **required** |
| `className` | `string` | - |
| `asChild` | `boolean` | - |

### ComboboxGroup

Groups related items. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `heading` | `string \| ReactNode` | - |
| `className` | `string` | - |
| `asChild` | `boolean` | - |

### ComboboxGroupLabel

Label for a group. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `asChild` | `boolean` | - |

### ComboboxEmpty

Displays content when no items match the filter. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `children` | `ReactNode` | `No results found.` |
