## API Reference

### Kbd

Styled representation of a single key (e.g. ⌘, Enter).

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `default`, `outline` | `default` |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |

### KbdGroup

Groups multiple keys into one shortcut (e.g. ⌘ + K).

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |
