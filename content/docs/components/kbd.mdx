```tsx lineNumbers
  Kbd, 
  KbdGroup

```

## API Reference

### Kbd

Keyboard shortcut key. Renders a `kbd` element.

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `default`, `outline` | `default` |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |

### KbdGroup

Container for grouping multiple keys into one shortcut. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |
