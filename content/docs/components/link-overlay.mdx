## Usage

Wrap a container (e.g. Card) with `LinkBox` and add `LinkOverlay` with an `href` to make the whole area a link.

```tsx
import { LinkBox, LinkOverlay } from "@/components/ui/link-overlay";
import { Card } from "@/components/ui/card";

<LinkBox>
  <Card>
    <LinkOverlay href="/blog/post-1" />
    <h2>Blog Post Title</h2>
    <p>Some content...</p>
  </Card>
</LinkBox>
```

Inner links (e.g. "Read more") work by using `relative z-10` so they stay above the overlay.

### With Next.js Link

Use `asChild` on both `LinkBox` and `LinkOverlay` to compose with `next/link`:

```tsx
import Link from "next/link";
import { LinkBox, LinkOverlay } from "@/components/ui/link-overlay";

<LinkBox asChild>
  <article>
    <LinkOverlay asChild>
      <Link href="/blog/post">Blog Post Title</Link>
    </LinkOverlay>
    <p>Content...</p>
  </article>
</LinkBox>
```

## API Reference

### LinkBox

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | - |

### LinkOverlay

| Prop | Type | Default |
|------|------|---------|
| `href` | `string` | required |
| `asChild` | `boolean` | `false` |
| `className` | `string` | - |
