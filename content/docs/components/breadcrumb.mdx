## Usage

```tsx
import {
  Breadcrumb,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbList,
  BreadcrumbPage,
  BreadcrumbSeparator,
} from "@/components/ui/breadcrumb";
```

```tsx showLineNumbers
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="/">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Current Page</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## API Reference

### Breadcrumb

Navigation landmark for the breadcrumb trail.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbList

Ordered list of breadcrumb segments.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbItem

Wraps a single breadcrumb link or page.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbLink

Link to a parent page in the hierarchy.

| Prop | Type | Default |
| --- | --- | --- |
| `href` | `string` | `-` |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbPage

Current page. Not a link; marked with `aria-current="page"` for accessibility.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbSeparator

Visual separator between segments. ChevronRight by default; customizable via children.

| Prop | Type | Default |
| --- | --- | --- |
| `children` | `React.ReactNode` | `<ChevronRight />` |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbEllipsis

Shows when middle segments are collapsed. MoreHorizontal icon by default; use with dropdown.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |
