## Usage

```tsx
import {
  Breadcrumb,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbList,
  BreadcrumbPage,
  BreadcrumbSeparator,
} from "@/components/ui/breadcrumb";

<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="/">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Current Page</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## API Reference

### Breadcrumb

Root element. Renders a `nav`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbList

Container for breadcrumb items. Renders an `ol`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbItem

Wrapper for each breadcrumb segment. Renders an `li`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbLink

Link for navigable breadcrumb items. Renders an `a`.

| Prop | Type | Default |
| --- | --- | --- |
| `href` | `string` | `-` |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbPage

Current page indicator. Renders a `span` with `aria-current="page"`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbSeparator

Visual separator between items. Renders an `li` with `ChevronRight` icon by default.

| Prop | Type | Default |
| --- | --- | --- |
| `children` | `React.ReactNode` | `<ChevronRight />` |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |

### BreadcrumbEllipsis

Ellipsis indicator for collapsed items. Renders a `span` with `MoreHorizontal` icon.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
| `asChild` | `boolean` | `false` |
