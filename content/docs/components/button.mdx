## Variants

You can change the variant of the button using the `variant` prop.

Available variants are: `solid`, `outline`, `success`, `info`, `warning`, `destructive`.

<ComponentVariants variants={['solid', 'outline', 'secondary', 'ghost', 'link', 'success', 'info', 'warning', 'destructive']} defaultValue="solid">

  </TabsContent>

## Sizes

You can change the size of the button using the `size` prop.

Available sizes are: `sm`, `md`, `lg`, `icon-sm`, `icon-md`, `icon-lg`.

<ComponentVariants variants={['sm', 'md', 'lg', 'icon-sm', 'icon-md', 'icon-lg']} defaultValue="md">
  <TabsContent value="sm">
    <Button size="sm">Small Button</Button>
  </TabsContent>

  <TabsContent value="md">
    <Button size="md">Medium Button</Button>
  </TabsContent>

  <TabsContent value="lg">
    <Button size="lg">Large Button</Button>
  </TabsContent>

  <TabsContent value="icon-sm">
    <Button size="icon-sm">
      <Plus />
    </Button>
  </TabsContent>

  <TabsContent value="icon-md">
    <Button size="icon-md">
      <Plus />
    </Button>
  </TabsContent>

  <TabsContent value="icon-lg">
    <Button size="icon-lg">
      <Plus />
    </Button>
  </TabsContent>
</ComponentVariants>

## Click Effect

You can change the click effect of the button using the `clickEffect` prop.

Available click effects are: `true`, `false`.

<ComponentVariants variants={['true', 'false']} defaultValue="true">
  <TabsContent value="true">
    <Button clickEffect={true}>Move 1px down when clicked</Button>
  </TabsContent>

  <TabsContent value="false">
    <Button clickEffect={false}>No click effect</Button>
  </TabsContent>
</ComponentVariants>

## Loading

You can show a loading indicator when the button is loading using the `isLoading` prop.

Available loading states are: `true`, `false`.

<ComponentVariants variants={['true', 'false']}>
  <TabsContent value="true">
    <Button isLoading={true}>Submit</Button>
  </TabsContent>

  <TabsContent value="false">
    <Button isLoading={false}>Submit</Button>
  </TabsContent>
</ComponentVariants>
