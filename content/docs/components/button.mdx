## Touch hitbox

The `.touch-target` utility extends the touch hitbox for better usability on touch devices.

Add a `.touch-target` utility to an element to extend the touch target.

```css
@utility touch-target {
  position: relative;
  &::before {
    content: "";
    position: absolute;
    display: block;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    width: 100%;
    height: 100%;
    min-height: 44px;
    min-width: 44px;
    z-index: 9999;
  }
}
```

## API Reference

| Prop | Type | Default | 
| --- | --- | --- | 
| `variant` | `default`, `outline`, `secondary`, `ghost`, `link`, `destructive` | `default` | 
| `size` | `xs`, `sm`, `md`, `lg`, `xl`, `icon-xs`, `icon-sm`, `icon-md`, `icon-lg`, `icon-xl` | `md` | 
| `clickEffect` | `boolean` | `true` | 
| `isLoading` | `boolean` | `false` |

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/button#api-reference).
