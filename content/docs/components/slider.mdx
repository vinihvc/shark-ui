## Installation

```bash
pnpm add @ark-ui/react
```

## Usage

```tsx
import { Slider } from "@/components/ui/slider";
```

The label element of the slider.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | - | Use the provided child element as the default rendered element. |

### Slider.ValueText

The value text element that displays the current value.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | - | Use the provided child element as the default rendered element. |

### Slider.HiddenInput

The hidden input element for form submission.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | - | Use the provided child element as the default rendered element. |
