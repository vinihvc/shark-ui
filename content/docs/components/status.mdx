## Usage

```tsx
import { Status } from "@/components/ui/status";

<Status.Root colorPalette="green">
  <Status.Indicator />
  Success
</Status.Root>
```

## API Reference

### Status.Root

| Prop | Type | Default |
|------|------|---------|
| `colorPalette` | `"gray" \| "red" \| "orange" \| "yellow" \| "green" \| "teal" \| "blue" \| "cyan" \| "purple" \| "pink"` | `"gray"` |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` |
| `className` | `string` | - |

### Status.Indicator

The status indicator dot. Inherits `colorPalette` and `size` from `Status.Root` context.
