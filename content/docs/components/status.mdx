## Usage

```tsx
import { Status } from "@/components/ui/status";

<Status.Root colorPalette="green">
  <Status.Indicator />
  Success
</Status.Root>
```

## API Reference

### Status

Renders a `span`.

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `"success" \| "info" \| "warning" \| "destructive"` | `"success"` |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` |
| `className` | `string` | - |
| `asChild` | `boolean` | `false` |

| Attribute | Description |
| --- | --- |
| `data-slot` | `status` |
