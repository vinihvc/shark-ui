## Usage

```tsx
import { Show } from "@/components/ui/show";

<Show when={isLoggedIn} fallback={<p>Please log in</p>}>
  <Dashboard />
</Show>
```

When `when` is truthy, children are rendered. Otherwise, `fallback` is rendered.

## API Reference

### Show

| Prop | Type | Default |
|------|------|---------|
| `when` | `boolean` | required |
| `fallback` | `React.ReactNode` | - |
| `children` | `React.ReactNode` | required |
