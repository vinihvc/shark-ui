## Usage

```tsx lineNumbers
import { 
  Table, 
  TableBody, 
  TableCaption, 
  TableCell, 
  TableHead, 
  TableHeader, 
  TableRow,
  TableFooter,
} from "@/components/ui/table";
```

```tsx lineNumbers
<Table>
  <TableCaption />
  <TableHeader>
    <TableRow>
      <TableHead />
      <TableHead />
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell />
      <TableCell />
    </TableRow>
  </TableBody>
  <TableFooter />
</Table>
```

## Accessibility

Always use a `<TableCaption />` to help assistive technologies announce what the table represents. 

Use with the `sr-only` class to make the caption screen-reader only if you don't want to show it.

```tsx lineNumbers
<Table>
  <TableCaption className="sr-only">
    Monthly revenue by product.
  </TableCaption>
  {/* ... */}
</Table>
```

## API Reference

### Table

Root wrapper. Renders a scrollable container with a native `table` element.

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `"plain"` \| `"striped"` | `"plain"` |
| `isHoverable` | `boolean` | `true` |
| `className` | `string` | `-` |

### TableHeader

Header section. Renders a `thead`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### TableBody

Body section. Renders a `tbody`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### TableFooter

Footer section. Renders a `tfoot`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### TableRow

Row element. Renders a `tr`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### TableHead

Header cell. Renders a `th`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### TableCell

Body or footer cell. Renders a `td`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |

### TableCaption

Table caption. Renders a `caption`.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | `-` |
