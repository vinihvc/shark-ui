## API Reference

### Badge

Root element of the badge. Renders a `span`

| Prop | Type | Default | 
| --- | --- | --- | 
| `variant` | `solid`, `secondary`, `outline`, `success`, `info`, `warning`, `destructive` | `solid` | 
| `size` | `sm`, `md`, `lg` | `md` | 
| `className` | `string` | `-` |
