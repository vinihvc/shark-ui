## Overview

Shark UI provides a collection of reusable components built on top of [Ark UI](https://ark-ui.com/) and styled with [Tailwind CSS](https://tailwindcss.com/). These components are designed to be accessible, customizable, and easy to use.

## Features

- **Accessible**: Built on Ark UI's solid foundation with proper ARIA attributes
- **Customizable**: Styled with Tailwind CSS and tailwind-variants
- **TypeScript**: Full TypeScript support with proper type definitions
- **Copy & Paste**: Easy to copy and paste into your projects
- **Consistent**: Follows design system principles

## Installation

You can install components individually or all at once:

```bash
# Install all components
pnpm add @ark-ui/react

# Or install specific components as needed
```

## Usage

Each component comes with:
- **Preview**: Interactive examples
- **Code**: Copy-paste ready code
- **Props**: Complete API documentation
- **Examples**: Real-world usage patterns

## Getting Started

1. **Choose a component** from the sidebar
2. **Preview** the component in different states
3. **Copy the code** and paste it into your project
4. **Customize** the styling to match your design system

## Customization

All components are built with Tailwind CSS and can be easily customized by:

- Modifying the `tailwind-variants` configuration
- Adding custom CSS classes
- Using CSS variables for theming
- Extending the component props
