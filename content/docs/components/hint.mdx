## Hint vs Tooltip

 - **Hint**: A lightweight, minimal tooltip that shows information on hover or focus.

- **Tooltip**: A more complex component that provides a more full-featured tooltip with a variety of options.

## API Reference

### Hint

Root element that provides context and wraps the trigger and content.

| Prop | Type | Default |
| --- | --- | --- |
| `side` | `"top" \| "bottom" \| "left" \| "right"` | `"top"` |
| `offset` | `string` | `"1.5"` |
| `className` | `string` | - |

### HintTrigger

The element that triggers the hint on hover or focus. Use `asChild` to merge props with a child component.

| Prop | Type | Default |
| --- | --- | --- |
| `asChild` | `boolean` | - |
| `className` | `string` | - |

### HintContent

The hint popup content displayed on hover or focus.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | - |
