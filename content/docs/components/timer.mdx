## Usage

```tsx
import { 
  Timer, 
  TimerArea, 
  TimerItem, 
  TimerSeparator, 
  TimerControl, 
  TimerActionTrigger,
} from "@/components/ui/timer";

<Timer targetMs={3600000} startMs={2400000}>
  <TimerArea>
    <TimerItem type="hours" />
    <TimerSeparator>:</TimerSeparator>
    <TimerItem type="minutes" />
    <TimerSeparator>:</TimerSeparator>
    <TimerItem type="seconds" />
  </TimerArea>
  <TimerControl>
    <TimerActionTrigger action="start">Start</TimerActionTrigger>
    <TimerActionTrigger action="pause">Pause</TimerActionTrigger>
  </TimerControl>
</Timer>
```

## API Reference

### Timer

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `targetMs` | `number` | `0` | The target time in milliseconds |
| `startMs` | `number` | `0` | The start time in milliseconds |

### TimerArea

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | `-` | The class name for the timer area |

### TimerItem

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `type` | `"hours"` \| `"minutes"` \| `"seconds"` \| `"days"` | `"seconds"` | The type of timer item |

### TimerSeparator

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | `-` | The class name for the timer separator |

### TimerControl

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | `-` | The class name for the timer control |

### TimerActionTrigger

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `action` | `"start"` \| `"pause"` \| `"resume"` | `"start"` | The action for the timer action trigger |

### TimerActionTriggerProps

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `action` | `"start"` \| `"pause"` \| `"resume"` | `"start"` | The action for the timer action trigger |

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/timer#api-reference).
