```tsx

  Tooltip, 
  TooltipContent, 
  TooltipTrigger 
} from "@/components/ui/tooltip";
```

```tsx showLineNumbers

  <TooltipContent>

  </TooltipContent>
</Tooltip>
```

## API Reference

### Tooltip

Root component that wraps the tooltip. Manages open state and positioning.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `positioning` | `PositioningOptions` | `{ placement: "top" }` | Options used to position the tooltip content |
| `openDelay` | `number` | `0` | Delay before opening the tooltip (ms) |
| `closeDelay` | `number` | `100` | Delay before closing the tooltip (ms) |
| `lazyMount` | `boolean` | `true` | Whether to enable lazy mounting |
| `unmountOnExit` | `boolean` | `true` | Whether to unmount on exit |
| `open` | `boolean` | — | The controlled open state |
| `defaultOpen` | `boolean` | — | The initial open state when uncontrolled |
| `onOpenChange` | `(details: OpenChangeDetails) => void` | — | Called when the open state changes |
| `disabled` | `boolean` | — | Whether the tooltip is disabled |
| `interactive` | `boolean` | `false` | Whether the tooltip content is interactive (stays open on hover) |
| `closeOnClick` | `boolean` | `true` | Whether to close on click |
| `closeOnEscape` | `boolean` | `true` | Whether to close when Escape is pressed |
| `closeOnPointerDown` | `boolean` | `true` | Whether to close on pointer down |
| `closeOnScroll` | `boolean` | `true` | Whether to close on scroll |

### TooltipTrigger

The element that triggers the tooltip on hover or focus. Renders a `button` by default.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `asChild` | `boolean` | — | Use the provided child element as the default rendered element |

### TooltipContent

The tooltip content displayed in a popover. Renders a `div` element.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | — | The class name for the content |
| `asChild` | `boolean` | — | Use the provided child element as the default rendered element |

### TooltipArrow

Optional arrow pointing to the trigger. Place inside `TooltipContent`.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `asChild` | `boolean` | — | Use the provided child element as the default rendered element |
