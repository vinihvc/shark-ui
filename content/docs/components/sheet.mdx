## Usage

```tsx lineNumbers
import { 
  Sheet, 
  SheetTrigger, 
  SheetContent, 
  SheetHeader,
  SheetTitle, 
  SheetDescription, 
  SheetBody,
  SheetFooter,
  SheetClose,
} from "@/components/ui/sheet";
```

```tsx lineNumbers
<Sheet>
  <SheetTrigger />
  <SheetContent>
    <SheetHeader>
      <SheetTitle />
      <SheetDescription />
    </SheetHeader>
    <SheetBody>
      {/** your content here */}
    </SheetBody>
    <SheetFooter>
      <SheetClose />
    </SheetFooter>
  </SheetContent>
</Sheet>
```

## API Reference

### Sheet

Root element of the sheet. Renders a `div`. Uses Ark UI Dialog under the hood.

| Prop | Type | Default |
|------|------|---------|
| `open` | `boolean` | `-` |
| `defaultOpen` | `boolean` | `false` |
| `onOpenChange` | `({ open }: OpenChangeDetails) => void` | `-` |
| `modal` | `boolean` | `true` |
| `closeOnEscape` | `boolean` | `true` |
| `closeOnInteractOutside` | `boolean` | `true` |
| `initialFocusEl` | `() => MaybeElement` | `-` |
| `finalFocusEl` | `() => MaybeElement` | `-` |
| `onEscapeKeyDown` | `(event: KeyboardEvent) => void` | `-` |
| `onInteractOutside` | `(event: InteractOutsideEvent) => void` | `-` |
| `className` | `string` | `-` |

### SheetTrigger

Button that opens the sheet. Renders a `button`.

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | `-` |

### SheetContent

Container for the sheet content. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `side` | `"right" \| "left" \| "top" \| "bottom"` | `"right"` |
| `variant` | `"default" \| "inset"` | `"default"` |
| `showCloseButton` | `boolean` | `true` |
| `className` | `string` | `-` |

### SheetHeader

Header container for the sheet. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `title` | `string` | `-` |
| `description` | `string` | `-` |
| `className` | `string` | `-` |

### SheetBody

Scrollable content container. Renders a `div` wrapped in a `ScrollArea`.

| Prop | Type | Default |
|------|------|---------|
| `scrollFade` | `boolean` | `false` |
| `className` | `string` | `-` |

### SheetFooter

Footer container for the sheet. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `className` | `string` | `-` |

### SheetTitle

Title element for the sheet. Renders an `h2`.

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | `-` |

### SheetDescription

Description element for the sheet. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | `-` |

### SheetClose

Button that closes the sheet. Renders a `button`.

| Prop | Type | Default |
|------|------|---------|
| `asChild` | `boolean` | `false` |
| `className` | `string` | `-` |

### SheetOverlay

Backdrop overlay for the sheet. Renders a `div`.

| Prop | Type | Default |
|------|------|---------|
| `className` | `string` | `-` |

### SheetPositioner

Positioning wrapper for the sheet content. Renders a `div`. Used internally by `SheetContent`; expose when composing custom layouts.

| Prop | Type | Default |
|------|------|---------|
| `side` | `"right" \| "left" \| "top" \| "bottom"` | `-` |
| `variant` | `"default" \| "inset"` | `"default"` |
| `className` | `string` | `-` |

---

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/dialog#api-reference).
