## API Reference

### Spinner

Use the `Spinner` component to display a loading indicator. It forwards SVG props and supports `className` for sizing and styling.

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | — |
| `aria-label` | `string` | `"Loading"` |
