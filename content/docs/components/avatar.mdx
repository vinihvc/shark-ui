<ComponentInstallation componentName="avatar">

Install the following dependencies:
  
```bash
pnpm add @ark-ui/react
```
</ComponentInstallation>

## Usage

```tsx lineNumbers
import { 
  Avatar, 
  AvatarImage, 
  AvatarFallback 
} from "@/components/ui/avatar";
```

```tsx lineNumbers
<Avatar>
  <AvatarImage src="https://github.com/vinihvc.png" />
  <AvatarFallback>VV</AvatarFallback>
</Avatar>
```

## API Reference

### Avatar

| Prop | Type | Default | 
| --- | --- | --- | 
| `size` | `sm`, `md`, `lg` | `md` | 
| `variant` | `success`, `info`, `warning`, `destructive` | `success` | 

### AvatarBadge

| Prop | Type | Default | 
| --- | --- | --- | 
| `variant` | `success`, `info`, `warning`, `destructive` | `success` |
