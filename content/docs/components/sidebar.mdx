## Usage

```tsx
import {
  Sidebar,
  SidebarContent,
  SidebarFooter,
  SidebarGroup,
  SidebarGroupContent,
  SidebarGroupLabel,
  SidebarHeader,
  SidebarInset,
  SidebarMenu,
  SidebarMenuButton,
  SidebarMenuItem,
  SidebarProvider,
  SidebarTrigger,
} from "@/components/ui/sidebar";
```

```tsx showLineNumbers
<SidebarProvider>
  <Sidebar>
    <SidebarHeader />
    <SidebarContent>
      <SidebarGroup>
        <SidebarGroupLabel>Navigation</SidebarGroupLabel>
        <SidebarGroupContent>
          <SidebarMenu>
            <SidebarMenuItem>
              <SidebarMenuButton asChild href="#">
                <span>Home</span>
              </SidebarMenuButton>
            </SidebarMenuItem>
          </SidebarMenu>
        </SidebarGroupContent>
      </SidebarGroup>
    </SidebarContent>
    <SidebarFooter />
  </Sidebar>
  <SidebarInset>
    <header>
      <SidebarTrigger />
    </header>
    <main>{children}</main>
  </SidebarInset>
</SidebarProvider>
```

## Structure

A Sidebar is composed of:

- **SidebarProvider** - Handles collapsible state and context
- **SidebarTrigger** - Button to toggle the sidebar
- **Sidebar** - The sidebar container
- **SidebarHeader** - Sticky header at the top
- **SidebarContent** - Scrollable content area
- **SidebarFooter** - Sticky footer at the bottom
- **SidebarGroup** - Section within the content
- **SidebarInset** - Main content area (use with `variant="inset"`)

## SidebarProvider

| Prop | Type | Default |
| --- | --- | --- |
| `defaultOpen` | `boolean` | `true` |
| `open` | `boolean` | `-` |
| `onOpenChange` | `(open: boolean) => void` | `-` |

## Sidebar

| Prop | Type | Default |
| --- | --- | --- |
| `collapsible` | `"offcanvas" \| "icon" \| "none"` | `"offcanvas"` |
| `placement` | `"left" \| "right"` | `"left"` |
| `variant` | `"sidebar" \| "floating" \| "inset"` | `"sidebar"` |

### Collapsible modes

- **offcanvas** - Sidebar slides off-screen when collapsed
- **icon** - Sidebar collapses to icons only
- **none** - Non-collapsible sidebar

## useSidebar

```tsx
import { useSidebar } from "@/components/ui/sidebar";

function MyComponent() {
  const { state, open, setOpen, toggleSidebar, isMobile } = useSidebar();
}
```

| Property | Type | Description |
| --- | --- | --- |
| `state` | `"expanded" \| "collapsed"` | Current sidebar state |
| `open` | `boolean` | Whether sidebar is open |
| `setOpen` | `(open: boolean) => void` | Set open state |
| `toggleSidebar` | `() => void` | Toggle sidebar |
| `isMobile` | `boolean` | Whether on mobile viewport |

## Keyboard shortcut

Press `⌘B` (Mac) or `Ctrl+B` (Windows) to toggle the sidebar.
