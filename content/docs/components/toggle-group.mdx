## Usage

```tsx showLineNumbers

  ToggleGroup, 
  ToggleGroupItem 
} from "@/components/ui/toggle-group"; 
```

```tsx showLineNumbers

  <ToggleGroupItem value="1">Item 1</ToggleGroupItem>
  <ToggleGroupItem value="2">Item 2</ToggleGroupItem>
</ToggleGroup>
```

## API Reference

### ToggleGroup

Root component. Provides shared variant and size to child toggles via context. Renders a `div`.

| Prop | Type | Default |
| --- | --- | --- |
| `variant` | `"outline"` \| `"ghost"` | `"ghost"` |
| `size` | `"sm"` \| `"md"` \| `"lg"` | `"md"` |
| `spacing` | `0` \| `1` \| `2` \| `3` \| `4` \| `5` \| `6` \| `8` \| `10` | `0` |
| `multiple` | `boolean` | `true` |
| `defaultValue` | `string[]` | - |
| `value` | `string[]` | - |
| `onValueChange` | `(details: ValueChangeDetails) => void` | - |
| `orientation` | `"horizontal"` \| `"vertical"` | `"horizontal"` |
| `disabled` | `boolean` | `false` |
| `className` | `string` | - |

For more props, see [Ark UI Toggle Group](https://ark-ui.com/docs/components/toggle-group#api-reference).

### ToggleGroupItem

Individual toggle button within the group. Inherits variant and size from parent ToggleGroup. Renders a [`<Toggle />`](/docs/components/toggle).

| Prop | Type | Default |
| --- | --- | --- |
| `value` | `string` | **required** |
| `disabled` | `boolean` | `false` |
| `className` | `string` | - |
| `aria-label` | `string` | - |

### ToggleGroupSeparator

Visual separator between toggle items. Use within `ToggleGroup` for a consistent styled divider. Renders a [`<Separator />`](/docs/components/separator).

| Prop | Type | Default |
| --- | --- | --- |
| `orientation` | `"horizontal"` \| `"vertical"` | `"vertical"` |
| `className` | `string` | - |

---

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/toggle-group#api-reference).
