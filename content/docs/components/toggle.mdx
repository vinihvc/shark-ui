## API Reference

### Toggle

Styled wrapper for the toggle with variant and size options.

| Prop | Type | Default |
|------|------|---------|
| `variant` | `"outline"` \| `"ghost"` | `"ghost"` | 
| `size` | `"sm"` \| `"md"` \| `"lg"` | `"md"` |
