## Installation

```bash
npm install @ark-ui/react
```

## Usage

```tsx
import { ClientOnly } from "@/components/ui/client-only";
```

```tsx
<ClientOnly>
  <div>This content only renders on the client.</div>
</ClientOnly>
```

## API Reference

### ClientOnly

Renders children only on the client side.

| Prop | Type | Default |
| --- | --- | --- |
| `fallback` | `ReactNode` | - |
| `children` | `ReactNode` | - |

---

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/utilities/client-only#api-reference).
