## Usage

Format provides three components for locale-aware formatting. All use the native `Intl` API and are cached for performance.

```tsx
import { Format } from "@/components/ui/format";

// Byte formatting (file sizes)
<Format.Byte value={120_000} />
// Renders: "120 KB"

// Relative time
<Format.RelativeTime value={new Date("2025-05-05")} />
// Renders relative to now: "in X days" or "X days ago"

// Number formatting
<Format.Number value={1234567} />
// Renders: "1,234,567"
```

You can also import individual components:

```tsx
import {
  FormatByte,
  FormatNumber,
  FormatRelativeTime,
} from "@/components/ui/format";
```

## API Reference

### Format.Byte

Formats byte values with automatic unit conversion. Extends `Intl.NumberFormat` for byte-specific formatting.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `number` | — | The byte size to format (required) |
| `unit` | `"byte"` \| `"bit"` | `"byte"` | The unit granularity to display |
| `unitDisplay` | `"long"` \| `"short"` \| `"narrow"` | — | How to display the unit |
| `unitSystem` | `"decimal"` \| `"binary"` | — | Use decimal (1000) or binary (1024) |
| `locale` | `string` | — | BCP 47 locale string |

### Format.RelativeTime

Formats dates as relative time using `Intl.RelativeTimeFormat`.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `Date` \| `number` | — | The date to format (required) |
| `numeric` | `"always"` \| `"auto"` | — | Numeric vs. verbal representation |
| `style` | `"long"` \| `"short"` \| `"narrow"` | — | Format style |
| `locale` | `string` | — | BCP 47 locale string |

### Format.Number

Formats numbers using `Intl.NumberFormat` with caching.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `number` | — | The number to format (required) |
| `locale` | `string` | — | BCP 47 locale string |
| `style` | `"decimal"` \| `"percent"` \| `"currency"` \| `"unit"` | `"decimal"` | Format style |
| `currency` | `string` | — | ISO 4217 currency code (e.g., `"USD"`) when `style="currency"` |
| `unit` | `string` | — | Unit for `style="unit"` (e.g., `"kilometer"`, `"liter"`) |
| `notation` | `"standard"` \| `"compact"` \| `"scientific"` \| `"engineering"` | `"standard"` | Number notation |
| `minimumFractionDigits` | `number` | — | Minimum decimal places |
| `maximumFractionDigits` | `number` | — | Maximum decimal places |
