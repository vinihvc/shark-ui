## What is Shark UI?

Shark UI is a opinionated UI library built on top of [Ark UI](https://ark-ui.com) and [shadcn](https://shadcn.com). It provides a set of components and utilities that are designed to be used in a modern web application. Shark is a combination of ```shadcn``` and ```ark``` initials.

## Motivation

[Ark UI](https://ark-ui.com) has more headless components, it's made by the team behind [Chakra UI](https://chakra-ui.com) also is a direct alternative to [Radix Primitives](https://radix-ui.com/primitives) and [Base UI](https://base-ui.com) and a more opinionated approach to [shadcn](https://shadcn.com).

## Solution

Because of the [Ark UI](https://ark-ui.com) docs don't have examples with [Tailwind CSS](https://tailwindcss.com) and how to create a reusable design system, I decided to create a new library.
