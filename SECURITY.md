# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in shark-ui, please report it responsibly.

**Do not** open a public GitHub issue for security vulnerabilities.

Instead, send details to **viniciusvicentini@live.com** with:

1. A description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Any suggested fixes (if you have them)

We will respond as soon as possible and keep you updated on the status of the report.

## Acknowledgments

We appreciate the security research community for helping keep shark-ui and its users safe.
